# SQL Marketing Analytics Portfolio

A hands-on SQL learning project built around real-world marketing analytics scenarios. This repository documents my journey from SQL beginner to writing production-style queries — organized by skill level and culminating in a full campaign performance analysis project.

---

## About This Project

I'm a marketing analytics professional (background in Datorama / Marketing Cloud Intelligence) returning to the field and building fluency in SQL as a core skill. This repo is a live record of that learning — rough drafts included.

The final goal is a polished **Marketing Campaign Performance Analysis** that answers real business questions about campaign ROI, channel efficiency, conversion trends, and audience performance.
